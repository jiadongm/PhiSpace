# 05_pdc_fibroblast_supp — pDC–fibroblast co-presence (Reviewer 2 C3)

Supplementary analysis added during the revision to address Reviewer 2's
question on whether pDC–fibroblast co-presence patterns differ between
healthy and lung cancer samples, with an independent-cohort validation in
CosMx data.

Conclusion (see `pDC/CLAUDE.md` for full results table): Visium across 40
samples shows all 5 fibroblast subtypes have significantly higher pDC
co-presence in tumour vs non-tumour (Wilcoxon p < 1e-3 to < 1e-6), with a
clear Healthy < Background < LUAD < LUSC gradient. CosMx confirms the
pattern is niche-dependent (Adventitial / Alveolar FB significant by KW).

- `${data_root}/Visium_NSCLC/allSamples/` — 40-sample Visium data (extends
  the 18-sample set used in the main Visium case study). See

- `${data_root}/LungRef/AzimuthLung2.0_sce_0.1sub.qs` — same Azimuth
  reference as `01_visium_case/`.
- `${data_root}/CosMx_lung/SCE_obj/allTissue_SCE.qs` — built by
  `02_cosmx_case/00_prepareSCE.R` + `01_runPhiSpace4Refs.R`. Run those
  first if you have not already.
- `${output_root}/pDC/output/PhiRes_allSamples.qs` — generated by step 01 below.

## Run order

```
00_prepareData_visium40.R    Load 40 Visium samples (.h5 + spatial/), build SCE list.
01_runPhiSpace_visium.R      Run PhiSpace (Azimuth Lung 2.0, 500 features) on all 40 samples.
02_visium_pDCFibroblast.R    Visium analysis: per-sample pDC vs 5 fibroblast subtype Spearman
                             correlations across 4 conditions (Healthy / Background / LUAD / LUSC).
03_cosmx_pDCFibroblast.R     CosMx validation: per-sample, per-niche pDC vs fibroblast Spearman
                             correlations on smoothed PhiSpace scores. Tests with Kruskal-Wallis.
```
