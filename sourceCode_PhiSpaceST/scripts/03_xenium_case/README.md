# 03_xenium_case — Xenium FFPE NSCLC (Section 2.4 / Fig 4)

PhiSpace ST applied to a 10x Xenium FFPE human lung cancer dataset (1 sample,
161,894 cells, Xenium Human Multi-Tissue and Cancer Panel of 377 gene
targets), using the same 4 Natri et al. lineage references as the CosMx case
study to enable cross-platform comparison. Added during the manuscript
revision (Reviewer 1 request to demonstrate Xenium / MERFISH platforms).

## Inputs (set via `config/paths_local.yml`)

Downloaded individually from the 10x Genomics CDN (the full Xenium Output
Bundle is ~tens of GB; we only need the 3 small files plus the supplemental
pathology GeoJSON). See `00_prepareData.R` for the exact `curl` commands.

- `${data_root}/Xenium_NSCLC/Xenium_V1_humanLung_Cancer_FFPE_cell_feature_matrix.h5` (6.4 MB)
- `${data_root}/Xenium_NSCLC/Xenium_V1_humanLung_Cancer_FFPE_cells.csv.gz` (8.4 MB)
- `${data_root}/Xenium_NSCLC/Xenium_V1_humanLung_Cancer_FFPE_gene_panel.json` (205 KB)
- `${data_root}/Xenium_NSCLC/Xenium_V1_humanLung_Cancer_FFPE_annotation.geojson`
  (pathology polygons, supplied by 10x — required for `01_assignPathology.py`)
- `${data_root}/LungFibrosis/scRNA-seq/{immune,epithelial,endothelial,mesenchymal}_sce.qs`
  (re-uses CosMx case study's references; build them with `02_cosmx_case/00_prepareSCE.R`
  if you have not already)

## Run order

```
00_prepareData.R                  Download files; build SCE; filter control probes; log1p; QC.
01_assignPathology.py             Map pathology GeoJSON polygons onto cells (point-in-polygon
                                  via shapely). Stored as colData(sce)$pathology_combined.
02_runPhiSpace4Refs.R             Run PhiSpace separately per lineage; concatenate (43 cell types).
03_nicheAnalysis.R                Spatial smoothing (k=15 Gaussian); PCA; multi-k k-means clustering
                                  (k=2..10) + Hungarian alignment; gene-expression clustering.
04_clusterEnrichment.R            Heatmaps of mean PhiSpace scores per cluster + PLS-DA per cluster.
                                  Generates Fig 4C (k=8 z-scored heatmap with pathology bar).
05_coPresenceAnalysis_k8.R        Per-niche Spearman co-presence matrices for k=8 + PCA of
                                  vectorised upper triangles. Generates Supp Fig S8 (A,B,C).
06_coPresenceAnalysis_multiK.R    Same analysis for k=4..10 (sensitivity check).
```

There is no `_utils.R` in this folder yet — the case study reuses utility
helpers from `scripts/02_cosmx_case/_utils.R` (or sources them inline). If you
add Xenium-specific helpers, save them as `_utils.R` here.

## Notes

- The pathologist-drawn GeoJSON contains 11 polygons across 5 categories;
  only the first Tumor polygon overlaps the Xenium FOV (the tissue strip is
  ~45 mm in y, but Xenium imaged only the first ~3.6 mm). The 5-category
  `pathology_combined` annotation is what is used in Fig 4A.
- `PhiClust_vs_GenClust.png` is generated by `03_nicheAnalysis.R` but is **not
  used in the manuscript**; it is retained as a supplementary diagnostic.
